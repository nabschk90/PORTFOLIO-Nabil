Projet Personnel - Application IA Locale (DeepSeek)

Le projet met en place une architecture complète (frontend, backend, moteur IA) pour offrir un assistant intelligent hors ligne pour mon usage personelle, sécurisé et personnalisable.

Fonctionnalités clés :

  • Exécution d’un modèle LLM DeepSeek en local via Ollama, sans connexion internet
  • Application multiplateforme développée avec Electron (HTML / CSS / JavaScript)
  • API REST avec FastAPI pour gérer la communication entre UI et moteur IA
  • Stockage local des historiques via SQLite pour traçabilité et continuité
  • Séparation claire (Frontend / Backend / IA) permettant l’évolution et l’ajout de modèles
  • Recommandations d'amélioration et priorisation des corrections
  • Aucune donnée envoyée sur des serveurs externes, traitement 100% local

🛡️ Infrastructure Security En cours de travail

STAGE Femto-ST - Système d’Alerte VPS

Mise en place d’un outil de supervision permettant de détecter les dégradations de performance sur des serveurs VPS (Ubuntu) et d’alerter automatiquement les utilisateurs. Le système analyse l’état de la machine et propose des procédures de remédiation afin de rétablir son bon fonctionnement.

Fonctionnalités clés :

  • Détection des facteurs impactant les VPS (CPU, RAM, stockage, services, logs)
  • Scan automatisé des machines pour repérer les problèmes de performance
  • Génération de recommandations pour restaurer l’état de santé du VPS
  • Envoi de notifications par email en cas de dégradation détectée
  • Séparation claire (Frontend / Backend / IA) permettant l’évolution et l’ajout de modèles
  • Monitoring régulier pour anticiper les incidents et maintenir la stabilité
  • Solution pensée pour les plateformes pédagogiques (Dept MMI / M2I)
Linux (Ubuntu VPS) Python / Bash (scripts de monitoring) Agent snmp Python / node.js

🛡️ Infrastructure Security

Projet Universitaire — Pentesting (SAE 3.04)

Réalisation d’une série de défis de cybersécurité orientés pentesting, couvrant plusieurs domaines : web, système, forensic et programmation. Le projet consiste à identifier, exploiter et comprendre des vulnérabilités réelles, tout en développant une méthodologie d’analyse et d’attaque.

Fonctionnalités clés :

  • Exploitation de vulnérabilités : NoSQL, LDAP, XSS, CSRF, Directory Traversal
  • Contournement d’authentification et récupération de données sensibles
  • Injection XSS, vol de cookies, manipulation de requêtes HTTP
  • Stockage local des historiques via SQLite pour traçabilité et continuité
  • Investigation de dump mémoire avec Volatility (Command & Control)
  • Recommandations d'amélioration et priorisation des corrections
  • Escalade de privilèges via scripts Bash et manipulation de variables d’environnement
Kali Linux Burp Suite Volatility HTTP / Web technologies Injection SQL
